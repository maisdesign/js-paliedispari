# Paliedispari

Piccolo progetto frontend che include due mini-giochi in JavaScript: **Pari o Dispari** e **Verifica Palindromo**. L'interfaccia usa Bootstrap e funziona direttamente in browser.

## Funzionalita
- Pari o Dispari: l'utente sceglie pari/dispari e un numero da 1 a 5; il computer genera un numero casuale e determina il vincitore.
- Verifica Palindromo: controlla se una parola o frase inserita e' un palindromo (ignorando spazi e punteggiatura).

## Come usare
1. Apri `index.html` nel browser.
2. Sezione "Gioco Pari e Dispari":
   - scegli pari/dispari;
   - inserisci un numero da 1 a 5;
   - premi "Gioca".
3. Sezione "Verifica Palindromo":
   - inserisci una parola o frase;
   - premi "Verifica".

## Struttura del progetto
- `index.html` interfaccia utente.
- `script.js` logica dei due giochi.
- `README.md` questo file.

## Requisiti
Nessuna dipendenza locale. Serve solo un browser moderno con JavaScript abilitato.

## Note
Il gioco Pari o Dispari valida l'input e mostra un messaggio se il numero non e' compreso tra 1 e 5.
